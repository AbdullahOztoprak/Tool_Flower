How to Generate and Use SSH Keys

        
    
          
  1. Open a terminal (or PowerShell on Windows).
    2. 
          
  2. Run: ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
    3. 
          
  3. Press Enter to accept the default file location.
    4. 
          
  4. When prompted, set a passphrase (optional, but recommended).
    5. 
          
  5. Your public key will be in ~/.ssh/id_rsa.pub (or C:\Users\\YourName\\.ssh\\id_rsa.pub on Windows).
    6. 
          
  6. Copy the contents of the public key file and paste it into the field above.
    7. 
          
  7. On your VM/server, add the public key to ~/.ssh/authorized_keys (create the file if it doesn't exist).
    8. 
        

        

          Tip: Never share your private key (id_rsa), only the public key (id_rsa.pub).
